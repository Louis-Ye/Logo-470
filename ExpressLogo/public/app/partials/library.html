forward / fd

Name:

Standard forward
Shortcut fd
Related back

Syntax:

forward number
fd number

What's this?

forward or fd means "move forward". The opposite command is back or bk. By using this command, the turtle will move forward for number pixels.

Example:

fd 50



back / bk

Name:

Standard back
Shortcut bk
Related forward

Syntax:

back number
bk number

What's this?

back or bk means "move backward". It is just the opposite to forward or fd. By using this command, the turtle will move backward for number pixels.

Example:

bk 60



left / lt

Name:

Standard left
Shortcut lt
Related right

Syntax:

left number
lt number

What's this?

left or lt means "turn left". The opposite command would be right or rt. By using this command, the turtle will turn left for number degrees.

Example:

lt 70
fd 50



right / rt

Name:

Standard right
Shortcut rt
Related left

Syntax:

right number
rt number

What's this?

right or rt means "turn right", which is the opposite to left or lt. By using this command, the turtle will turn right for number degrees.

Example:

rt 90
fd -50



make

Name:

Standard make

Syntax:

make "variable expression

What's this?

make can be used to assign a value to a variable, and you may used this variable in the following program by writing a ":" mark followed by the variable name. It may be hard to understand what is so called variable. See the examples below and hopefully you will know the power of a variable.

Example:

make "A 30 fd :A



if

Name:

Standard if
Related else

Syntax:

if condition [commands]

What's this?

if is a very classical conditional statement. You can see this statement in almost every programming language. The commands would be executed only if the condition is true. What if your want to execute other commands when the condition is false? See else and you will get the answer.

Example:

fd 30
make "A 1
if :A>0 [lt -90]
bk -40



else

Name:

Standard else
Related if

Syntax:

if condition [commands] else [commands]

What's this?

else can not be used individually without if statement. By using else statement, you can assign commands to execute not only when the condition is true, but also when it is false.

Example:

bk 50
make "A 1
if :A>2 [rt 60] else [lt 60]
fd 50



repeat

Name:

Standard repeat

Syntax:

repeat number [commands]

What's this?

repeat in Logo can be considered as a simple version of for statement in other languages. The commands would be executed for number times. Although any real number can be accepted as the number parameter by the compiler. You are still encouraged to use a none negative integer.

Example:

repeat 36
[
    lt 10
    fd 10
]



penup / pu

Name:

Standard penup
Shortcut pu
Related pendown

Syntax:

penup
pu

What's this?

penup or pu is used to "pick up" the turtle from the canvas. After using this command, the following movement of the turtle will not be drawn on canvas, although the turtle will still move follow the commands. What if you want to "put down" the turtle? Use pendown!

Example:

make "A 0
repeat 36
[
    lt 10
    fd 10
    if :A>20 pu
    make "A :A+1
]



pendown / pd

Name:

Standard pendown
Shortcut pd
Related penup

Syntax:

pendown
pd

What's this?

pendown or pd is used to "put down" the turtle onto the canvas after penup has been used. You do not need to initialize the canvas because the turtle is "down" by default.

Example:

make "A 0
repeat 36
[
    lt 10
    fd 10
    if :A==0 [pd] else [pu]
    make "A 1-:A
]



penwidth / pw

Name:

Standard penwidth
Shortcut pw

Syntax:

penwidth number
pw number

What's this?

Find the lines too thin? Set the width for the lines by yourself using penwidth. The following lines would be drew in the width of number pixels.

Example:

make "A 0
repeat 36
[
    lt 10
    fd 10
    if :A==0 [penwidth 1] else [penwidth 5]
    make "A 1-:A
]



color

Name:

Standard color

Syntax:

color [number number number]
color [3-byte RGB code]

What's this?

Do not like the boring black turtle and lines? You can set them to your own colours using color command. The three numbers represent the RGB value (from 0 to 255) of the following lines. The 3-byte RGB code should be write in term of 3 hexadecimal number (66CCFF, for example).If you do not know what is RGB and what is hexadecimal number, you can just pick one colour by clicking the "color" button under the canvas.

Example:

make "A 0
repeat 36
[
    if :A==0 [color [255 0 0]] else [color [0 255 255]]
    lt 10
    fd 10
    make "A 1-:A
]



clearscreen / cs

Name:

Standard clearscreen
Shortcut cs

Syntax:

clearscreen
cs

What's this?

Command clearscreen / cs is for clear the canvas. All the lines that has been drawn will be cleaned. But neither the turtle would return "home", nor the background colour would be cleaned.

Example:

make "A 0
repeat 36
[
    if :A==20 [clearscreen]
    lt 10
    fd 10
    make "A :A+1
]



home

Name:

Standard home
Related setxy

Syntax:

home

What's this?

By using home command, you can move the turtle back to the centre of the canvas and reset the direction it is pointing to to upward direction. Notice that if penup is not used before home, a lined will be drew from the original position to the "home".

Example:

repeat 18
[
    lt 10
    fd 10
]
home



setxy

Name:

Standard setxy
Related home

Syntax:

setxy number number

What's this?

setxy is not a standard Logo command, so we do not encourage you to use it in your program. Abuse of setxy can destroy both your code and your picture. But if you find it very important to you. Just feel free to use it. setxy can be regarded as a special version of home. The two numbers represent the coordinate position of the destination on canvas (the (0,0) is at the left-bottom of the canvas). Please notice that unlike home, the direction of the turtle will not be reset.

Example:

make "A 0
repeat 36
[
    if :A==18 [setxy 250 250]
    lt 10
    fd 10
]
home



to & end

Name:

Standard to
Standard end

Syntax:

to function :parameter1 :parameter2 ...
    commands
end

function number1 number2 ...

What's this?

The to and end sentence can define a function in the program. You may wonder what is a function? Well, it is quite hard to explain. As a beginner to programming, you may only need to know that by using this tricky thing, you no more need to write those duplicated code. You can just put them into a function and "call" that function every time you need it.

Oh, wait. I just forgot to tell you how to use the function. Before you can call a function, you must define that function just using to & end sentence. Please remember that a function name can only be used once. You may notice that there is some parameters following after the function. They are just the variable I introduced in the make section. You can assign different values to those parameters to achieve different results. There is no limitation on the number of parameters. Which means you can define as much parameters as you want. Of course, you may also leave it blank if you do not need any. To call that function, just do like you use other sentences. write the function followed by some numbers or variables to set those parameters. By the way, the number of the objects you give should be exactly equal to the number of parameters that you have defined, or there will be an error.

If you are still confused, why not just look at the example?

If you are a beginner, you can just skip the following information:

The parameters are formal parameters they are just the copy of the origin. So make change of these parameters would not change the original object.

You can call functions inside a function. It can be other functions, or just the function itself if you want to solve a problem in a recursive way.

You can define a new function inside a function. Since all the functions are global, it is not meaningful to do so.

However, the variables declared in a function is local. It can not be used outside that function and would be deleted when the function is end. Even if you call the same function again, the value would not be inherited.

Example:

to MyFlower :DEG :LEN
    lt :DEG
    fd :LEN
    if :DEG<360
    [
        home
        MyFlower :DEG+10 :LEN
    ]
end
MyFlower 0 70
