Draw a straight line:

forward 50 || fd 50

Try more simple commands:

Backward move 50 pixels: backword 50 || bk 50
Turtle left turns 90 degrees: left 90 || lt 90
Turtle right turns 90 degrees: right 90 || rt 90

Repeat commands

With repeat commands, you dont have to type same commands again and again.
Try these codes: repeat 36 [lt 10 fd 10]

Make variables

Now you can draw a cute circle! :D
With variable, you can draw more transformable shape in a simple way.
Try these codes:
make "angle 5
repeat 36 [
  lt :angle
  fd 10
  make "angle :angle - 1
]

Control Flow with if/else

Programming is all about controlling the commands.
Use if/else to control the behavior of this little turtle!
Try these codes:
make "count 0
repeat 36 [
  if :count < 18 [
   lt 10 fd 5
  ]
  else [
   rt 10 fd 5
  ]
  make "count :count + 1
]

Create your own command - using function

With function definition, you can create your own command
For example, try defining a function and use it to draw a circle!
Try these codes:
to drawCircle
  repeat 36 [lt 10 fd 10]
end

drawCircle

Use function with parameters

Try these codes:
to drawCircle :r
  repeat 36 [lt 10 fd 5 + :r]
end

make "i 5
repeat 5 [
  drawCircle :i
  make "i :i - 1
]


Concept of local variables and global variables

What is local variable and global variables?
  • Global variables are the variables you make at the top level of the program.
  • Local variables are the variables you make inside a scope.
    • What is scope?
    Scope is a range between certain codes. There are two situations that can be seen as scope:
  • Space between [ ... ]
  • Space between to ... end
    • Followed by these features:
  • Local variable which was firstly maked inside a scope can not be seen outside of this scope.
  • Parameters of a function are local variables belonging to the scope between to ... end
  • Global variable can be seen any where after its make.
    • These codes will give errors:
    if 2 > 1 [
	make "A 2
]
fd :A   ;  ':A' is not visible outside of the scope it was firstly made!
			
    to func :B
	make "B 3
end
fd :B   ;  ':B' is not visible outside of the scope of the function body!

    Recursively call function

    Recursive calling is a very important concept in programming.
    It means that you can call the function itself inside the body of its own definition!
    Try these codes:
    to drawCircle
      repeat 36 [lt 10 fd 10]
      drawCircle
    end

    drawCircle

	lt 60 
	line :count2 :length

    Colorful Spiral

    
to spiral :w :a :x :c :ww
if :c == 0 [ pu pd ]
else [
	penwidth :ww
	color [ 255 - :w * 2 0 :w * 2 ]
	lt :x
	fd :w
	pu
	bk :w
	rt :x
	fd :w
	pd 
	rt :a
	spiral :w + 1 :a :x + 0.7 :c - 1 :ww + 0.1
]
end

home cs
spiral 1 30 10 90 1

    Maze

    
to l :l :w
if :l == 0 [pd]
else [
	rt 90 
	r :l - 1 :w
	fd :w lt 90
	l :l - 1 :w
	fd :w
	l :l - 1 :w
	lt 90 fd :w
	r :l - 1 :w
	rt 90
]
end

to r :l :w
if :l == 0 [pd]
else [
	lt 90
	l :l - 1 :w
	fd :w rt 90
	r :l - 1 :w
	fd :w
	r :l - 1 :w
	right 90
	fd :w
	l :l - 1 :w
	lt 90
]
end
